All-In-One YouTube downloader with automatic audio conversion.
---
## Requirements
* git
* ffmpeg in path
## Setup
Run these commands to get started:
```
git clone https://github.com/phorcysed/ytdl
npm i
```
Then run ``ytdl <video url>`` and it will download it!
